## Resumo do que o plugin faz
- O plugin descreve uma skill de design de front-end que foca em gerar **codigo pronto para producao** com **direcao estetica ousada e coerente**, evitando uma estetica generica.
- O README cita exemplos de pedidos em linguagem natural (dashboard, landing page, settings).

## Principios centrais (parafraseado)
- Antes de codar, entender contexto e **assumir uma direcao estetica clara**.
- Dar prioridade a:
  - **Tipografia** (escolhas com personalidade; hierarquia consistente).
  - **Cor & tema** (paleta dominante + acentos; consistencia via tokens/variaveis).
  - **Motion** (animacoes de impacto + micro-interacoes; reduzir motion quando preferido).
  - **Composicao** (layouts menos previsiveis, mas intencionais).
  - **Background/detalhes** (profundidade/atmosfera; texturas/padroes/sombras).
- Evitar padroes genericos recorrentes (fontes e layouts "default", paletas timidazinhas, etc.).
